# Task 1: Data Immersion & Wrangling

## Objective
To acquire, clean, and prepare a real-world dataset for analysis by identifying and resolving data quality issues such as missing values, duplicates, and inconsistent formatting.

## Dataset
A public e-commerce sales dataset was used to simulate a real business scenario, as no internal dataset was provided.

## Key Activities
- Created a detailed data dictionary describing each variable
- Performed data quality assessment (missing values, duplicates, inconsistencies)
- Cleaned and transformed the dataset using Python (Pandas)
- Engineered new features such as Customer Age and Order Month
